## OWLCMS4 Cloud Installer 4.7.4-beta2

This release is for deploying OWLCMS4 to the Heroku cloud.  See [this page](https://jflamy-dev.github.io/owlcms4-prerelease/#/index) for the general documentation.

To install: click on this button.  You will be prompted to create a (free) account on the Heroku cloud if you do not have one already.

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy?template=https://github.com/jflamy-dev/owlcms4-heroku-prerelease/tree/4.7.4-beta2)

After the initial installation, you can use the [OWLCMS Cloud Updater](https://github.com/owlcms/owlcms4-heroku-updater/) program to update you cloud deployments while keeping your settings.
